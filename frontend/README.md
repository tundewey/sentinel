# Sentinel Frontend

Next.js Pages Router dashboard with Clerk authentication.

## Required Env
Use `frontend/.env.local`:
- `NEXT_PUBLIC_API_URL`
- `NEXT_PUBLIC_CLERK_PUBLISHABLE_KEY`
- `CLERK_SECRET_KEY`

## Routes
- `/` incident input + analysis
- `/dashboard` incident history + job lookup
- `/sign-in` Clerk sign-in
- `/sign-up` Clerk sign-up
